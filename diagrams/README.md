## Chèn vào Word

1. Mở `KLTN_Tran_Viet_Duc_P2P_Chat.docx`
2. Tìm sơ đồ ASCII tương ứng → xóa
3. **Insert → Pictures** → chọn PNG
4. Caption: *Hình X.X — [tên]*

## Chỉnh sơ đồ chính xác hơn

Paste file `.mmd` vào [mermaid.live](https://mermaid.live) → Export PNG.
